触发载荷解析失步,整图崩到 ~0.01;③ 裸传 ≤7dB 反输给 ① 均匀,10dB 起才收敛到 ② 理想。

极小带宽即可在设计 SNR 填平,但依赖信道质量估计,失配时再悬崖。约束:回复中文;CIFAR-100
级训练可接受、ImageNet 级不可;GPU 跑前先与用户约定时长/样本。

## Goals / Non-Goals

**Goals:**
- 同 b392/ρ=0.25 预算下把垂直悬崖变成**有底斜坡**:高 SNR 达 ②,SNR 下降时平滑退化,
  极低 SNR 最坏退化到**基础层(1-bit)质量**,**绝不出现③式失步崩盘(跌到随机猜)**。
  诚实声明:该底可能低于 2-bit 均匀 ①(分层把半数比特押在增强层的代价);A1 价值在
  消除灾难崩盘与失配鲁棒,不在底部绝对值。
- 信道失配场景(按固定 SNR 设计保护、真实 SNR 更差)下 UEP 再悬崖而 A1 不崩。
- v1 零训练跑通。

**Non-Goals:**
- 不追求设计 SNR 点准确率优势(A1≈UEP≈②)。
- 不追求开销优势(A1 与 UEP 半斤八两)。
- 不上 LDPC/Turbo(只用重复码);不做 ImageNet 级训练;v1 不做同步标记/修复头。

## Decisions

**D1. 两层切分 M_base=1(基础 196 + 增强 196)。** b392 平均每 patch 2 bit,两层都要非空
只有这一种切法。备选 M_base=2 把预算吃光、无增强层,排除。

**D2. 不传 M_i 图,两端各自派生。** 收发两端在同一份基础层重建上跑 ViT → 派生逐 bit 相同
的 ΔM_i,增强层边界自动对齐。这是删除失步机理的核心。备选(传 M_i + FEC)即 EEP/UEP,降为
对照基线。

**D3. 注意力量化到 16 档锁一致性。** 两端浮点前向可能微小漂移导致排序翻转 → ΔM_i 错位;
把 a_i 量化到 16 档使漂移落进同档。备选(整数/定点推理)工程量过大;备选(用基础层残差能量
代替注意力)偏离论文 attention-based 卖点。实现前用纯 CPU 脚本实测漂移以确认档位。

**D4. 基础层用重复码轻 FEC。** 基础层是唯一命门,逐 bit 无误则增强层自动对齐。重复码速率
1/r,BER 闭式,便于精确分预算;同一砌块供 EEP/UEP 对照复用。

**D5. 预算口径=同信道带宽(口径 B)。** 六线对齐同实传比特数,FEC 膨胀计入总预算。理由:
图2 失配对决必须公平,不让任何一方靠多占带宽虚胜。诚实声明:A1 不省带宽。

## Risks / Trade-offs

- [1 bit 基础层算注意力质量不足 → 派生分配次优、达不到 ②] → 实现第一步做注意力一致性
  验证实验(原图 a_i vs base 重建 a_i 的派生分配重叠度);不足时退路:(i) 换工作点
  b588/ρ=0.375 让 base 升 2 bit;(ii) 加 CIFAR-100 级小修复头;(iii) 原图注意力蒸馏。
- [浮点漂移大于预期、16 档仍不一致] → 验证脚本先量漂移,必要时加粗档位或定点关键算子。
- [base 残余误码导致两端 ΔM_i 偏移] → v1 靠重复码 r 压到极低 BER;v2 再加周期同步标记兜底。
- [同带宽口径下 A1 因基础层均匀损失分配自由度,设计点略亏 UEP] → 接受;A1 卖点是失配
  鲁棒性非设计点准确率,已在 Non-Goals 声明。
